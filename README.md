# MirroHaus — Landing Page (Next.js)

Single-page site for MirroHaus with **About Us** and **Our Work** sections.

## Quick start
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this folder to GitHub (e.g. `mirrohaus/mirrohaus-site`).
2. On Vercel, **Add New → Project → Import Git Repository**.
3. Accept the defaults (Next.js detected) and click **Deploy**.

The site will build and go live at `*.vercel.app`. Add your custom domain in **Settings → Domains**.
